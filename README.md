# SEEEEDS
 Stop your players from breaking plants that are not ready to be harvested.

## Build instructions:

### Linux

On Linux, this can be built like any [Maven](https://maven.apache.org/) project.

Make sure to have the `maven` package installed, and run

```
$ mvn
```
